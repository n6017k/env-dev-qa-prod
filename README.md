# env-dev-qa-prod

Minimal Node (Express) server demonstrating dev/devops flows.

Quick start

1. Install dependencies:

```powershell
npm install
```

2. Start server:

```powershell
npm start
```

3. Endpoints:

- `/` - basic JSON greeting
- `/health` - health status
- `/env` - NODE_ENV

Run tests:

```powershell
npm test
```
